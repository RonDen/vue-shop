# vue-shop

#### 介绍

摸鱼六佬的前端项目，学习构建电商后台管理系统前端网站

学习目标：入门前端 vue 开发，了解前后端分离技术。

![](img/front-back-sep.png)

项目结构

```
$ tree .
vue-shop
├─code  项目代码
├─docs  文档、学习资料、任务计划等
└─img   需要的图片
```

#### 功能模块划分

![](img/arch.png)

1. 登录/退出功能
2. 用户管理模块
3. 权限管理模块
4. 分类管理模块
5. 参数管理模块
6. 商品管理模块
7. 订单管理模块
8. 数据统计模块
